[Grimoire.js] d3.js を使って作品を並べてみるテスト

Grimoire.js を使用したサンプル（約90本）を並べてみました。

＜使い方＞
・サムネイルをクリックするとサンプルが背景に表示されます。
　シフトキー + クリック … 作品を別ウインドウで表示
　Ctrlキー + クリック … 作品紹介ページを別ウインドウで表示

＜対応した点＞
・Force Layout（力学モデルでグラフ描画するレイアウト）にて描画するよう対応
・サンプルを iframe を使って背景に表示するよう対応
