マックロソフト Windous XP をシミュレーションしてみた。

＜対応した点＞
・Clippy.js を用いて、キャラクターの表示を再現。
・空耳辞書＋Speak.js により音声読み上げに対応。（Firefox / Chrome にて対応。IE は未対応。）

＜対応できていない＞
・本当はイルカ（カイル君）を使いたかったが Clippy.js のキャラクターとして用意されていないようだった為、かわりにインコ（Peedy）を使用。
