[WebGL] three.js で glTF 2.0形式のデータを表示してみるテスト（その８改）

Project "Polly" は glTF 2.0 の新機能をすべて紹介する為に制作されたものらしいです。

＜対応しようとした点＞
・project_polly.gltf で定義されているカメラを使用するよう対応。

2017/11/18 Three.js r89 + モデル更新。一部スキニングアニメーションの問題が解消されていることを確認。
2017/10/10 Three.js r88
