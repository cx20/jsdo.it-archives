[WebGL] lightgl.js でリサージュ図形を描いてみるテスト

lightgl.js は、WebGL API の冗長な記述をうまく吸収してくれる為 2D Canvas のような手軽さで 3D 描画が行えるようです。

＜対応した点＞
・lightgl.js の OpenGL immediate mode を使用してリサージュ図形を描くよう対応。
