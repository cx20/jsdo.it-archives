[WebGL] XenoGL + OimoPhysics.js を試してみるテスト

XenoGL は kfurumiya さんによる WebGL 2.0 用ライブラリです。
MatrixGL は kfurumiya さんによる行列演算ライブラリです。
OimoPhysics.js は saharan さんによる物理演算ライブラリです。

＜対応した点＞
・XenoGL + OimoPhysics.js でキューブを物理演算させるよう対応。
