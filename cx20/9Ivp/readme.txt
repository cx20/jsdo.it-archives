SVG で関数従属性図の説明を書いてみるテスト

情報処理試験に出てくる図を SVG で表現してみた。

＜対応した点＞
・SVG を使って、図を書いてみた。
・使用したタグ：marker / polygon / rect / line / text

＜対応できなかった点＞
・文字の縦書き機能を試そうとしたが、ブラウザによって実装されていないようなので取りやめた。
