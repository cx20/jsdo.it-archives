WebRuby を自動コーディングしてみるテスト

組み込み Ruby の「mruby」を JavaScript 化した「webruby」というのがあるらしいので、使ってみました。
サイズが1.4MB と、多少大きかったので、PNG 化の手法を用いて圧縮し jsdo.it のリソースとして利用しています。

＜対応した点＞
・WebRuby.js を PNG 化することにより、ライブラリサイズを 1.4MB→563KB に圧縮
・CodeMirror で Ruby コード色分けを使用
