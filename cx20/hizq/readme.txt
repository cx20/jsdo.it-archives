p2.js でドット絵を物理演算してみるテスト（その１）

3D 物理演算ライブラリ「cannon.js」の作者 schteppe さんによる
2D 物理演算ライブラリ「p2.js」を試してみました。

＜対応した点＞
・「p2.js Canvas Box example」をベースに、複数の四角形をドット絵になるように配置。
