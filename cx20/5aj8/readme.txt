WebGL Earth API を試してみるテスト

WebGL Earth はオープンソースの 3D 地球儀で内部的に CesiumJS のエンジンをレンダリングに使用しているようです。
また、地図操作に関する API は「Leaflet」（2D 地図表示用ライブラリ）の API に近いものになっているようです。

＜試してみた点＞
・２つのサンプル（Satellite + Animation）を合成してみた。
