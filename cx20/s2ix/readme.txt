[WebGL] SpectorJS を試してみるテスト（three.js編）

SpectorJS は WebGL/WebGL2 に対応した汎用的な WebGL キャプチャツールです。Chrome 拡張版と JS 組み込み版があるようです。
ここでは JS 組み込み版を使ってみました。赤いボタンをクリックするとキャプチャが開始されます。フルスクリーン推奨です。

＜使用方法＞
<script src="spector.bundle.js"></script>

var spector = new SPECTOR.Spector();
spector.displayUI();
