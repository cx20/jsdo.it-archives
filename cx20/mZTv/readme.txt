[WebGL] WebGL Utils を試してみるテスト

WebGL Utils は gl-matrix.js でもお馴染みの Tojiro さんによる、軽量かつ標準的な WebGL Utilities です。
似たような名前のライブラリが多々あるので使用する際は、他と混同しないよう注意ください。

＜対応した点＞
・WebGL Utils を用いて、三角形を描くよう対応。

＜WebGL Utilities で提供しているコード一覧＞
・wglu-debug-geometry.js
・wglu-preserve-state.js
・wglu-program.js
・wglu-stats.js
・wglu-texture.js
・wglu-url.js
