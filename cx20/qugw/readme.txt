[WebGL] three.js で glTF 2.0形式のデータを表示してみるテスト（その８）

Project "Polly" は glTF 2.0 の新機能をすべて紹介する為に制作されたモデルとのことです。

＜対応した点＞
・project_polly.gltf を表示させるよう対応。

＜変更履歴＞
2018/05/18 Three.js r93 + 環境マップを追加。
2017/11/18 Three.js r89 + モデル更新。一部スキニングアニメーションの問題が解消されていることを確認。
2017/10/09 Three.js r88
