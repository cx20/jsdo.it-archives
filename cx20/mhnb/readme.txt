空耳メーカー by Clippy.js

・Speak.js    … Chrome / Firefox のみ対応。IE では動作しません。
・Google TTS  … IE でも再生可能です。

＜対応した点＞
・「空耳メーカー」による読み上げ Clippy.js にて表現するよう変更
・Agent(※)を「Peedy」（インコ）に変更
　※ clippy.load() の箇所を変更することで、Agent を変更可能です。
 　　例）Bonzi, Clippy, F1, Genie, Genius, Links, Merlin, Peedy, Rocky, Rover
