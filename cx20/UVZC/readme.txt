[WebGL] Babylon.js + SolidParticleSystem を試してみるテスト

Solid Particle System (SPS) は jbousquie さんによる BabylonJS 拡張です。
SPS は Babylon.js v2.3 にて本体に組み込まれる予定のようです。

＜対応した点＞
・jerome さんの Solid Particle System 動作サンプルを jsdo.it で動作するよう移植。
