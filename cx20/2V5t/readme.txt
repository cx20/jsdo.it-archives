MMLEmitter で MML を試してみるテスト で MML を試してみるテスト

「MMLEmitter(※)」は「Web Audio API用のMMLイベントシーケンサー」とのことです。
※ ライブラリ名称が「wamml」→「MMLEmitter」に変更されました。

＜対応した点＞
・Playボタン押下時にテキストエリアの内容を再生するよう対応。

＜対応できていない点＞
・Firefox だと、ノイズが入るっぽいです。
・IE は、現時点では Web Audio API に未対応の為、実行できません。

＜変更履歴＞
2014/08/20 ライブラリ名変更「wamml」→「MMLEmitter」に伴う対応
